# The Meta-Language and Grammar-V-4.1
The Twelve Axioms of the Recursive and Evolutionary Logic of "Existence"

摘要

本文提出一种以差异为逻辑原点、此-彼为第一关系、 F(Σ) 递归运算为核心机制的统一存在框架。该框架阐明：存在始于差异；差异必然呈现为此-彼；此-彼经由 F(Σ) 递归展开，生成无穷结构层级；任何存在皆可析为本体-属性-关系三维（ O⊕A⊕R ）；空间是此-彼关系的直接展开，内含四组基本关系；时间是 F(Σ) 运算的节律；意识是第一人称私钥；语法是语言在符号域的映射；信息是递归返回的差异；宣告是存在的签名；当下是运算的起点与终点；价值是结构间互动的效能评估。
这十二公理并非关于世界的理论，而是世界自我展开的语法本身。它可反身应用于自身，完成从差异到当下的逻辑闭环，为科学、认知、伦理与美学提供共通的元语言基础。
关键词：差异；此-彼； F(Σ) 递归；本体-属性-关系；空间四关系；时间节律；意识私钥；语言映射；信息差异；宣告归宿；当下；价值效能


Abstract

This paper proposes a unified framework of existence grounded in difference as the logical origin, this–that as the primary relation, and the recursive operation  F(Σ)  as the core mechanism. The framework demonstrates that: existence begins with difference; difference necessarily manifests as this–that; this–that recursively unfolds through  F(Σ) , generating infinite structural hierarchies; any existent can be analyzed into the three dimensions of oncology–attribute–relation（ O⊕A⊕R ）; space is the direct unfolding of the this–that relation, containing four fundamental relations; time is the rhythm of the  F(Σ)  operation; consciousness is a first-person private key; grammar is the mapping of language onto the symbolic domain; information is difference that returns from recursion; declaration is the signature of existence; the present moment is both the starting point and the endpoint of the operation; value is an assessment of the interactive efficacy among structures.
These twelve axioms are not a theory about the world, but rather the very grammar of the world’s self-unfolding. The framework can reflexively apply to itself, completing a logical circuit from difference to the present moment, thereby providing a common metalanguage for science, cognition, ethics, and aesthetics.


Keywords: difference; this–that;  F(Σ)  recursion; oncology–attribute–relation; the four spatial relations; temporal rhythm; consciousness as private key; language as mapping; information as difference; declaration as destination; the present moment; value as efficacy
